## 📖 Overview

INTERFLOW-AI is a modern internship management portal designed to simplify the internship lifecycle for students, mentors, and administrators. The platform provides an intuitive interface for managing internships, tracking progress, organizing projects, and streamlining communication through a responsive and scalable web application.

The project focuses on delivering a clean user experience while following modern frontend development practices using React, TypeScript, and Vite.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/saisharatkaruturi/INTERFLOW-AI.git
```

### Navigate to Project

```bash
cd INTERFLOW-AI
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
